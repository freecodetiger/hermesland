# Hermes Protocol

Hermes Protocol is the shared contract between Gateway, Swift SDK, Island UI,
and smoke tests. The MVP transport is a JSON message stream.

## Server Event Envelope

Every server event is an `EventEnvelope`:

```json
{
  "event_id": "evt_001",
  "seq": 1,
  "type": "message.accepted",
  "created_at": "2026-01-01T00:00:00.000Z",
  "payload": {
    "message_id": "msg_001",
    "client_msg_id": "client_msg_001"
  }
}
```

Required fields:

- `event_id`: stable idempotency key for client-side dedupe.
- `seq`: user-scoped monotonic integer. It must be a non-negative safe integer.
- `type`: one of the event types listed below.
- `created_at`: ISO 8601 timestamp created by the event producer.
- `payload`: event-specific object.

## Event Types

Message events:

- `message.accepted`
- `message.delta`
- `message.completed`
- `message.failed`

Task events:

- `task.started`
- `task.progress`
- `task.completed`
- `task.failed`
- `task.cancelled`
- `task.requires_approval`

Notification and agent events:

- `notification.created`
- `agent.online`
- `agent.offline`

System events:

- `system.error`

## Client Commands

Clients send command objects with a `type` and `payload`.

### message.send

`message.send` starts a user message flow. `payload.client_msg_id` is required
and is generated by the client so Gateway can correlate retries and the
resulting `message.accepted` event.

```json
{
  "type": "message.send",
  "payload": {
    "client_msg_id": "client_msg_001",
    "text": "Start the smoke test."
  }
}
```

### events.ack

`events.ack` tells Gateway which events the client has processed. Clients may
send explicit event ids, a `through_seq`, or both.

```json
{
  "type": "events.ack",
  "payload": {
    "event_ids": ["evt_001", "evt_002"],
    "through_seq": 2
  }
}
```

## Example Events

```json
{
  "event_id": "evt_002",
  "seq": 2,
  "type": "message.delta",
  "created_at": "2026-01-01T00:00:00.000Z",
  "payload": {
    "message_id": "msg_001",
    "delta": "Hello"
  }
}
```

```json
{
  "event_id": "evt_010",
  "seq": 10,
  "type": "task.requires_approval",
  "created_at": "2026-01-01T00:00:00.000Z",
  "payload": {
    "task_id": "task_004",
    "approval_id": "approval_001",
    "prompt": "Allow file write?",
    "actions": ["approve", "deny"]
  }
}
```

```json
{
  "event_id": "evt_011",
  "seq": 11,
  "type": "notification.created",
  "created_at": "2026-01-01T00:00:00.000Z",
  "payload": {
    "notification_id": "notif_001",
    "title": "Task completed",
    "body": "Smoke test passed."
  }
}
```

```json
{
  "event_id": "evt_014",
  "seq": 14,
  "type": "system.error",
  "created_at": "2026-01-01T00:00:00.000Z",
  "payload": {
    "error": {
      "code": "UNHANDLED_ERROR",
      "message": "Unexpected gateway error."
    }
  }
}
```

The full executable example set is exported from
`@hermesland/hermes-protocol` as `eventExamples` and `commandExamples`.
